Alongside this project, an outside example can be seen in my Analysis of Run Time repository where I use the skills there to improve my search algorithm for this database. The skills I learned in my DriverPass repository helped plan the project from where it was in development to where I wanted it to be. In my classes I have yet to collaborate as a team, however I have participated in GameJams in which I collaborate with randomly chosen fellow developers to create a final product in the form of a game.
