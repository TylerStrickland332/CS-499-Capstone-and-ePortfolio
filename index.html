Improvements

The improved code was improved with the three prior mentioned focus areas in mind. Below are the explanations of how each area was improved, the challenges I faced with each, and how the I did/didn't achieve my expected course outcomes.

Link to improvement repository is linked above.

Software Design and Engineering: I implemented the use of the home device's calendar so that it has the possibilty to be used offline, as opposed to Google API's.

Algorithms and Datastructures: Introduction of UserID numbers to make securing data more effective, increasing security and ensuring you don't lose your data if you change your username.

Databases: Use of userIDs to improve search capabilities of user data that could greatly cut down search time in large-scale deployment of the app.
